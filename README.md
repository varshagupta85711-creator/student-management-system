# student-management-system
The Student Management System is a web-based application developed using HTML, CSS, and JavaScript to simplify the management of student records and academic information.
The system provides separate login portals for students and administrators. Students can securely access their profile, ERP ID, attendance records, and examination marks. Administrators have the ability to add, update, delete, and manage student records, monitor attendance, and maintain academic data efficiently.
The application uses LocalStorage to store data within the browser, ensuring data persistence without requiring a separate database. The user-friendly interface and responsive design make it easy to access and manage information across different devices.

-:Key Features:-

1.Student Login and Admin Login  
 2.Student Profile Management
 3.ERP ID Management
 4.Attendance Tracking
 5.Marks and Result Management
 6.Add, Edit, and Delete Student Records
 7.Secure Authentication System
 8.Browser LocalStorage Data Persistence
 9.Responsive and User-Friendly Interface

-:Technologies Used:-

1.HTML5
 2.CSS3
 3.JavaScript
 4.Browser LocalStorage

-:Objective:-

The main objective of this project is to digitize student record management, reduce manual paperwork, improve data accuracy, and provide quick access to academic information for both students and administrators.
